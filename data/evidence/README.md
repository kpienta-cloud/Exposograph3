# Evidence Layer

Evidence assertion records backing ExposoGraph 3.0 causal promotions and module parameters. `evidence_assertions.json` consolidates the real PMID/DOI/PMCID identifiers already attached to promoted causal edges (Phase 4) and module evidence bundles (Phase 3). Identifiers are copied from the ExposoGraph 2.0 provenance data — none are fabricated. Phase 4 evidence-governed promotion uses these assertions as the support test for moving registry edges into the causal layer.
